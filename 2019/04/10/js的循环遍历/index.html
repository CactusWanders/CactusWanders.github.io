js的循环遍历

CactusWanders
2019-04-10

while

当符合while条件的时候执行，语句可能不会执行

do…while

先执行，再检查while条件是否符合。语句至少会被执行一次

for

for…in

遍历对象所有可枚举的属性。

for…of

遍历迭代器，Iterator 接口主要供for…of消费。
原生具备 Iterator 接口的数据结构如下。
Array
Map
Set
String
TypedArray
函数的 arguments 对象
NodeList 对象

Array.map

遍历数组，不可中断

Array.forEach

遍历数组，不可中断

Array.filter

遍历数组，不可中断

Array.some

遍历数组

Array.every

遍历数组

Array.reduce

遍历数组

Array.reduceRight

遍历数组，相较于Array.reduce，从右向左计算
